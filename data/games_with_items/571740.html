Early Access Game

Get instant access and start playing; get involved with this game as it develops.

Note: This Early Access game is not complete and may or may not change further. If you are not excited to play this game in its current state, then you should wait to see if the game progresses further in development. Learn more

What the developers have to say:

Why Early Access?

“This is our first game release - so we want to make sure to have a working product before we call it a complete product. Additionaly we would like to include the players feedback in the game aswell as adding additional content until we and you are happy. Content Updates will still happen after the full-release.”

Approximately how long will this game be in Early Access?

“We don't expect to be too long in Early Access and would guess that a couple of months should be enough. Most of the features should be ready in the Early Access version and we mostly need to polish the game from the player feedback and add VR-Support.”

How is the full version planned to differ from the Early Access version?

“Additonal Content, Bug Fixes and basic VR-Support.”

What is the current state of the Early Access version?

“The game is fully functional and just lacks some polish.”

Will the game be priced differently during and after Early Access?

“We currently have no plans on increasing the price.”

How are you planning on involving the Community in your development process?

“The Community is a vital part of this game. We need the players feedback to polish the game and to understand what the community wants. We will try to be as open as possible during the development and active on the forums and other social media platforms.”

Yet Another Update

Represent your Team in Golf It!



Country Flags

For each team perticipating the soccer world cup we've implemented a flag. Flags will drop like other uncommon items and can be equiped as trail.

In case you've missed it

We've made several small patches and hotfixes since 0.5.4. Here is a short overview, in case you've missed it:

- User Experience

Our usability efforts slowly bear fruit, as we continue to improve the UI & UX. This time we've strongly enhanced the "How To Play" section, by adding 36 video tutorials made by TheStreamLife. Another 20+ new tutorials will be added soon. Thanks again Zach for doing an awesome job.

- Custom Sounds

If custom music isn't enough you can now add custom sounds. They could be used like sound effects and also be modified by the Eventsystem.

- Golfball Settings

Once again we've enhanced our Eventsystem by enabling the access to golfball settings. When your golfball triggers an event its physics now can get altered, strokes can be added or your clubs can get changed.

April 18

Update 0.5.4

Skins and more, in Golf It!



Skins

You can now change the skin of your golf ball - as soon as you got a new one. Skins are legendary items, thus the rarest. If you aren't lucky enough, don't despair, we've also implemented an item exchange system (inside the customization menu) for redundant or unused items. You can easily scrap them to get coins, which can then be used to buy new items.



As you can see, some skins will still interact with the color you've chosen.

Unlimited Players

Allowing only 8 players to join a lobby was designed as a softcap to prevent network issues. The underlying codebase, however, is build up completely dynamic. Therefore we've decided to remove the cap and implement a "private" mode instead, to ensure that only your friends can join, without entering a password.

User Interface

In order to increase the usability of Golf It we've finally started to improve our UI. This update includes a revamp of our "Host Game" UI, which improves the overview over your subscribed maps and provides additional options to sort them. Furthermore we've added a new UI for excahnging items and slightly improved the "Search Game" UI.

What's coming next

We're working on a new theme for quite a while now, which consums a lot of time, since we are creating assets which are already optimized for our editor. We can't provide any estimation about the release date, but it's going to be huge. We've already created more than 200 unique assets, which will allow to build more beautiful and complex scenes.

Our Discord

Christian & Jonas

About This Game

Golf It! is a multiplayer Minigolf game with focus on a dynamic, fun and creative multiplayer experience. One of the most exciting features is a Multiplayer Editor, where you can build and play custom maps together with your friends.

This version of the game features at least 5 different maps with 18 holes each: Grassland, Winterland, Graveyard, Mines and Pirates Cove. Every map has unique gameplay features and different aesthetics. Every asset of each area and more will be available for you inside the Multiplayer Editor and you are free to mix them all together to create the map of your desires! Currently there are over 1.000 placeable objects inside the Multiplayer Editor!

To have a more fun and different experience each time you play, we are using a ball hitting system which is based on the speed of the mouse. The faster you swing your mouse the harder you will hit the ball. It takes more experience and practice to hit that precious hole in one every time.
